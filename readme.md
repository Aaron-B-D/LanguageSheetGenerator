A simple Python application that allows you to easily generate language sheets in TypeScript with an optional associated enumeration.

Its functionality is loosely based on Godot’s multilingual text management and is designed for web projects.

Please keep in mind that this was one of my first complex projects in Python, so there may be some bugs.
